## 🎯 Ключевые цели (реализованы)

1.  **Надежное сохранение**: Данные сохраняются при закрытии/скрытии вкладки.
2.  **Синхронизация вкладок**: `BroadcastChannel` + `storage` event.
3.  **Дедупликация записей**: Сохранение только при реальном изменении данных.
4.  **Защита от гонки записей**: Механизм `updatedAt` + `_sourceId`.
5.  **Централизация**: Вся логика в хуке `useDayAutosave`.
