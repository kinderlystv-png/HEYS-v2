# Системные досье

Этот раздел предназначен для глубокого описания поведения, а не для списка
файлов. Каждое досье ведёт только к ключевым точкам входа; остальные детали
агент находит непосредственно в коде.

Статусы ниже относятся именно к проверке в новом контуре справочника:
`проверено` — критические утверждения подтверждены; `частично` — материал
полезен, но проверен не весь заявленный охват; `источник` — существующий
документ ещё не проходил новую ревизию; `пробел` — отдельного досье пока нет.
