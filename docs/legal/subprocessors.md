# Подпроцессоры (subprocessors) — HEYS

**Действует с:** 2026-05-20 **Версия:** 1.0

Список третьих лиц, которым HEYS передаёт персональные данные пользователей для
оказания услуг сервиса. Согласно 152-ФЗ ст.6 и GDPR Art.28.

## 1. Yandex.Cloud (LLC «Яндекс.Облако», Россия)

**Что обрабатывает:**

- PostgreSQL Managed Service: всю БД клиентов (профиль, дневник, согласия,
  подписки).
- Object Storage (S3-совместимый): резервные копии, статические ассеты PWA и
  лендинга.
- Cloud Functions: серверные API-обработчики (heys-api-rpc, heys-api-auth,
  heys-api-payments, heys-api-leads, heys-bot-client, heys-cron-\*).
- Lockbox: секреты (пароль БД, токены ботов).
- CDN: доставка статики лендинга/PWA.

**Регион:** Россия (ru-central1). **Юр.основание:** ст.18.5 152-ФЗ (хранение ПДн
россиян в РФ). **Контракт:** публичный договор оферты Yandex.Cloud.

## 2. ЮMoney / ЮKassa (НКО «ЮMoney», Россия)

**Что обрабатывает:**

- Платёжные транзакции (имя, телефон, email клиента, сумма, план).
- Возвраты средств.
- 54-ФЗ чеки (электронная касса).

**Регион:** Россия. **Юр.основание:** 54-ФЗ + договор с НКО. **Карточные
данные** обрабатываются на стороне ЮKassa (PCI DSS), HEYS их не хранит.

## 3. Telegram (Telegram Messenger Inc., США/Великобритания)

**Что передаётся:**

- **Через бота-куратора** (`TELEGRAM_CLIENT_BOT_TOKEN`): сообщения от клиента к
  куратору и обратно. Содержание — то что пользователь добровольно отправил.
- **Через бота-уведомлений** (`TELEGRAM_BOT_TOKEN`): аггрегаты по
  лидам/security-alerts (без PII в основном payload; UUID-prefix вместо имени).

**Юр.основание:** пользователь явно согласился использовать Telegram как канал
коммуникации (выбор messenger в TrialForm). **Caveat:** Telegram-сервера могут
находиться вне РФ — при выборе Telegram как канала пользователь информируется
(privacy-policy.md §10).

## 4. SMS-провайдер

**Статус:** не используется. Согласия фиксируются через checkbox + аудит-лог
(IP, User-Agent, версия документа). Cloud function `heys-api-sms` удалена
2026-05-22; код-заглушки в кодовой базе оставлены на случай возврата.

## 5. Аналитика

- **Yandex.Metrika** (LLC «Яндекс», Россия) — анонимная статистика посещений
  лендинга. Webvisor выключен. Cookies — после accept в CookieInfoBanner.
- **Google Analytics 4** — **удалён** 2026-05-20 (раньше был под env-флагом,
  трансграничная мина).
- **Meta Pixel** — **удалён** 2026-05-20 (та же причина).
- **Sentry / Mixpanel / другая трекинг-аналитика** — отсутствует.

## 6. AI/ML внешние API

**Отсутствуют.** В кодовой базе нет вызовов OpenAI/Anthropic/других AI-API с PII
в prompt'ах.

## Уведомление при изменении

Если добавляется новый подпроцессор:

1. Обновить этот документ.
2. Если новый подпроцессор обрабатывает PII вне РФ — bump версии `privacyPolicy`
   в
   [apps/landing/src/config/legal-versions.ts](../../apps/landing/src/config/legal-versions.ts)
   → запустится re-consent flow для всех активных клиентов (см.
   `check_required_consents_v2` + `ConsentOutdatedBanner`).
3. При трансграничной передаче — уведомление РКН (ст.12 152-ФЗ).
