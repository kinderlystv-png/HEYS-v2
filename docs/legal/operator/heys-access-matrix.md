# HEYS · Матрица доступов к ПДн

Статус: R0 working matrix. Базовый ownership-check подтверждён по SQL
2026-06-14; полный read-audit и break-glass остаются R0/R2 gap.

| Роль                     | Доступ                                   | Можно                                                               | Нельзя                                                                               | Контроль                             | Статус |
| ------------------------ | ---------------------------------------- | ------------------------------------------------------------------- | ------------------------------------------------------------------------------------ | ------------------------------------ | ------ |
| Оператор / ИП            | Все данные HEYS в объёме целей обработки | поддержка, администрирование, РКН/запросы субъектов                 | выгружать без цели, хранить секреты/ПДн в чате                                       | audit logs, секреты вне репо         | 🟡     |
| Назначенный куратор      | Только назначенные клиенты               | вести дневник, смотреть профиль/дневник/аналитику, отвечать клиенту | смотреть чужих клиентов, копировать базу в личные таблицы, слать ПДн в открытые чаты | client scoping, договор, лог доступа | 🟡     |
| Подменный куратор        | Временно назначенные клиенты             | подмена на период отсутствия                                        | постоянный доступ без назначения                                                     | временное назначение, отзыв доступа  | 🟡     |

| Система/бот              | Минимальные события                      | передавать `lead_id`, source, статус                                | слать телефон/ФИО/health-data в Telegram notification                                | smoke notification без ПДн           | 🟡     |
