## Чеклист

| Слой       | Статус | Проверка                                                   |
| ---------- | ------ | ---------------------------------------------------------- |
| Код/БД     | ⬜     | новые поля ПДн внесены в `heys-data-register.md`           |
| Логи       | ⬜     | health-values не попадают в audit/funnel/payment metadata  |
| Лендинг    | ⬜     | тексты не обещают лечение/диагностику/медрезультат         |
| Согласия   | ⬜     | версии документов совпадают с кодом и архивами             |
| РКН        | ⬜     | цели, категории, получатели и ИС соответствуют факту       |
| Кураторы   | ⬜     | доступ только к назначенным клиентам                       |
| Интеграции | ⬜     | нет неучтенной трансграничной передачи                     |
| Gaps       | ⬜     | открыты задачами в `маркетинг/22`, решения записаны в `15` |
