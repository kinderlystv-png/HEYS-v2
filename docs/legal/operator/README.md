## Юридическая основа

- ст. 18.1 152-ФЗ — обязанность оператора иметь локальный документ-политику
- ст. 22 152-ФЗ — обязанность подать уведомление в РКН (освобождение по ч. 2 НЕ
  действует, т.к. обрабатываются специальные категории: здоровье в HEYS, данные
  несовершеннолетних в Kinderly)

> Приказ о назначении ответственного за организацию обработки ПДн (ст. 22.1
> 152-ФЗ) для ИП-одиночки не требуется — ответственным по умолчанию является сам
> ИП.

## Что делать дальше

1. Открыть [rkn-notification-heys.md](rkn-notification-heys.md), сверить
   даты/контакты, идти на portal.rkn.gov.ru → подать уведомление
2. То же для Kinderly
3. Распечатать положение, подписать, хранить вместе с уставными документами ИП
4. Сохранить регистрационные номера из РКН, добавить в публичные документы
   (privacy-policy раздел 1.2)
