# Протокол реализации: задача на выбранные даты

## Результат

Статус: готово к локальному QA.

- Во вкладке списка задач добавлен переключатель «Повторять в выбранные даты».
- Мультикалендарь принимает даты из разных месяцев.
- Для каждой даты создаётся связанный слот одной задачи с общим
  `recurrenceGroupId`.
- В раскрытом слое задаются время и вид слота: обычный или фоновый; для фонового
  доступен цвет.
- Пустой набор дат блокирует создание и возвращает пользователя в календарь.

## Проверка

- `node --check` для модулей задач и календаря — пройдено.
- `planning-task-matrix.test.js` — пройдено, включая сортировку, дедупликацию и
  фоновые параметры слотов.
- `pnpm docs:reference:check` — пройдено.
- Scoped legacy bundle `postboot-3-ui-lazy` — пересобран для локального QA.
- Локальный full-stack runtime уже работает на `http://localhost:3001` и
  `http://localhost:4001`.

## Ревью

Проверить основной сценарий: включить повтор, выбрать даты в двух месяцах,
выбрать обычный или фоновый вид, создать задачу и убедиться, что все слоты
ссылаются на одну задачу. Вспомогательные действия «готово» и «драфт» намеренно
недоступны в режиме выбранных дат, чтобы не создавать уже завершённую серию.
