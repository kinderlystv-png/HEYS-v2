## 1. Роль (важно для легала)

**Научный советник / валидатор контента и методологии — НЕ лечащий врач в
продукте.** Куратор и продукт остаются «не медицинской услугой»; советник:

- валидирует методологию и клиент-видимый контент (лендинг, посты, FAQ) на
  корректность и отсутствие мед-заявлений;
- помогает держать границу «сопровождение/привычки, не лечение» (см. `10` §3.2,
  `COPY_VOICE`);
- усиливает duty-of-care протокол по РПП (стыковка с `23` §5);
- по явному письменному согласию — даёт право использовать имя/регалии в
  маркетинге (доверие на лендинге, и это же закрывает риск закона о рекламе
  услуг нутрициологов без медобразования — `10` §3.2).

> Советник **не ведёт клиентов, не ставит диагнозы, не назначает лечение** в
> рамках HEYS — иначе сами создаём «медуслугу», от которой открещиваемся.

## 2. Требования к кандидату

| Критерий         | Значение                                                                         |
| ---------------- | -------------------------------------------------------------------------------- |
| Образование      | **Высшее медицинское** (обязательно)                                             |
| Специализация    | Диетология / эндокринология / гастроэнтерология — желательно                     |
| Доп. компетенция | **Понимание РПП** (расстройств пищевого поведения) — сильный плюс (duty of care) |
| Опыт             | Практика + способность писать/проверять научно-популярный текст без жаргона      |
| Тон              | Разделяет позицию «бережно, без -N кг, без overpromise»                          |

---

## 3. Формат сотрудничества

- **Загрузка:** ~4–8 часов/мес (валидация контента + консультации по
  протоколам).
- **Оплата (вилка, утвердить):** ~3 000–5 000 ₽/час **или** фикс ~15 000–40 000
  ₽/мес.
- **Имя в маркетинге:** только по **явному письменному согласию**, с правом
  отзыва.
- **Оформление:** НПД/договор на услуги (не трудовой; см. рамку НПД `10` §3.1).
- Без overpromise кандидату: честно — стартап на Фазе 0, ограниченный объём,
  возможен рост по мере запуска.

## 4. Каналы поиска (минимум 3)

1. **Профильные медвузы / кафедры диетологии** (напр. Сеченовский, РНИМУ) —
   преподаватели и ординаторы, ищущие подработку/проекты.
2. **profi.ru / верифицированные врачи-диетологи** — с проверкой диплома.
3. **Проф-сообщества и конференции** по диетологии/нутрициологии
   (Telegram-группы врачей, отраслевые ассоциации).
