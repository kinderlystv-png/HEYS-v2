## S — Сильные стороны

| #   | Сильная сторона                                       | Почему это важно на рынке РФ                                                                                                                                   |
| --- | ----------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| S1  | **Связка «продукт + человек + протоколы»**            | Трудно копируется: трекеры не имеют операционки кураторов, нутрициологи 1:1 — приложения. Защищённая позиция в «дыре» рынка.                                   |
| S2  | **Анти-срыв как ядро ценности**                       | Бьёт в реальную боль («срываюсь, не выдерживаю рутину»), которую DIY-трекеры игнорируют. Сильный нерв для креатива.                                            |
| S3  | **Цена Pro 7 990 ₽ vs 25 000+ ₽ за нутрициолога 1:1** | Понятный ценностный разрыв: «то же сопровождение втрое дешевле + приложение».                                                                                  |
| S4  | **Зрелый продукт для триала**                         | Trial Machine v3, Insights v5, Health Score (41 паттерн), переезд в Yandex Cloud. Не MVP — можно запускать первые триалы; ЮKassa подключается до первых оплат. |
| S5  | **152-ФЗ compliance из коробки**                      | Данные в РФ, ПЭП для health-data, согласия. Снимает юридический риск и даёт аргумент доверия премиум-аудитории.                                                |
| S6  | **Бесплатный триал Pro без карты**                    | Сильнейший анти-фрикшн оффер: клиент проверяет ценность человека до оплаты.                                                                                    |
| S7  | **Дисциплина копирайта (`COPY_VOICE.md`)**            | Серьёзный, доказательный тон попадает в премиум-сегмент, отстраивает от крикливых марафонов.                                                                   |
| S8  | **Модель кураторов на НПД**                           | Масштабирование без раздувания ФОТ; кураторы — переменные затраты, привязанные к выручке.                                                                      |

## W — Слабые стороны

| #   | Слабая сторона                                     | Риск / следствие                                                                                                                                                          | Что делать                                                                                                                  |
| --- | -------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| W1  | **Экономика внешнего куратора при цене Pro 7 990** | Фикс 7 500 ₽/клиент (из брифа, под старую цену 12 990) при 7 990 ₽ = маржа ~0. ⚠️ **Снято на Фазе 0** (куратор = основатель, цена прибыльна сразу); актуально для Фазы 1. | Фаза 1: поднять цену под зарплату куратора (~9 900–10 900 ₽) — листы «Фазы-цена» и `11_Фазовое_ценообразование_и_запуск.md` |
| W2  | **Триал упирается в живого куратора**              | Каждый бесплатный триал = реальные часы человека → дорого и не масштабируется линейно. Всплеск трафика убивает SLA.                                                       | Жёсткий лимит слотов триала = ёмкость кураторов; список ожидания; найм опережает трафик                                     |
| W3  | **Высокий порог входа по доверию**                 | Премиум + health-data: клиент не покупает по первому касанию. Длинный цикл.                                                                                               | Контент-прогрев, отзывы, кейсы, прозрачность SLA                                                                            |
| W4  | **Зависимость от качества каждого куратора**       | Услуга = человек. Один слабый куратор = отток и плохие отзывы.                                                                                                            | Стандарты, шаблоны, контроль метрик (время до 👍/✅, NPS день 3/7)                                                          |
| W5  | **Нет бренда / нулевая известность**               | Холодный трафик дорогой; органики пока нет.                                                                                                                               | Контент-маркетинг + посевы + работа с первыми адвокатами                                                                    |
| W6  | **Узкая команда / ручная операционка**             | Соло-разработка; маркетинг и найм кураторов конкурируют за время.                                                                                                         | Приоритизация каналов (1–2, не 6), частичный аутсорс посевов                                                                |
| W7  | **Триал-абьюз и неактивные пользователи**          | Бесплатный Pro привлекает «халявщиков», жгущих ресурс куратора.                                                                                                           | Анти-абьюз уже частично есть (1 проба/телефон, критерии активного триала); добавить заморозку неактивных                    |

## O — Возможности

| #   | Возможность                                                        | Как использовать                                                                              |
| --- | ------------------------------------------------------------------ | --------------------------------------------------------------------------------------------- |
| O1  | **Тренд ЗОЖ: рынок услуг 270 млрд ₽, спрос на нутрициологов +34%** | Попутный ветер спроса — ловим тёплый интент в Директе и контентом                             |
| O2  | **Уход Instagram-рекламы освобождает бюджеты конкурентов**         | Перетекание в Telegram/VK/Директ — но и наш шанс занять Telegram-нишу контентом раньше других |
| O3  | **Дискредитация марафонов** (вред здоровью, разоблачения)          | Позиционирование «безопасная системная альтернатива марафонам» — отстройка на доверии         |
| O4  | **Пул кураторов растёт** (бум обучения на нутрициологов)           | Дешёвый доступ к кадрам + потенциальный B2B2C: кураторы приводят своих клиентов на платформу  |
| O5  | **Грант Yandex.Cloud для стартапов (до 2 млн ₽)**                  | Покрывает инфраструктуру — высвобождает кэш на маркетинг                                      |
| O6  | **Реферальная механика**                                           | Премиум-клиенты с результатом — лучший канал; заложить реф-программу                          |

## T — Угрозы

| #   | Угроза                                                                                            | Митигировать                                                                                                |
| --- | ------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- |
| T1  | **Регуляторика рекламы ужесточается** (Instagram-бан, ERID, сбор 3%, штрафы 100–500 тыс. юрлицам) | Процесс ERID + ответственный; никакой Instagram-рекламы; юр-ревью креативов                                 |
| T2  | **Регуляторика health/мед-заявлений**                                                             | Жёстко держать `COPY_VOICE` запреты: не «вылечу/похудеете гарантированно», только «помогаю выстроить режим» |
| T3  | **Демпинг марафонов и бесплатных трекеров**                                                       | Не конкурировать ценой; конкурировать удержанием и безопасностью                                            |
| T4  | **Накрутки в Telegram (35–40% каналов)**                                                          | Закупки только с проверкой статистики; performance-метрики, не охваты                                       |
| T5  | **Зависимость от ЮKassa / Yandex Cloud / SMS.ru**                                                 | Резервные провайдеры платежей и SMS в бэклог                                                                |
| T6  | **Выгорание/отток кураторов**                                                                     | Бонусы за удержание и SLA уже в модели; взаимоподмена; разумная загрузка                                    |

## Стратегический вывод из SWOT

**Играем S1+S2+S3 (уникальная связка, анти-срыв, ценовой разрыв) против
сегмента, недовольного марафонами (O3) и переплачивающего нутрициологу.**

Но **до платного трафика обязательно закрыть W1 (экономика куратора) и T1
(ERID-процесс)** — это два «красных» блокера. Рост трафика жёстко
синхронизировать с ёмкостью кураторов (W2): маркетинг здесь не «лей больше», а
«лей ровно столько, сколько кураторы вытянут с SLA ≤5% нарушений».
