# HEYS — привлечение и удержание кураторов (supply-сторона)

> Второй проход, supply-слой. Дата: 2026-06-09. Это **узкое место роста**: в
> модели «приложение + живой человек» масштаб ограничен не бюджетом на трафик, а
> числом качественных кураторов. Маркетинг спроса без supply = сорванный SLA и
> отток.

---

## 1. Почему это главный приоритет, а не «потом»

- Каждый платящий Pro-клиент требует доли времени куратора; норма — **14 Pro на
  куратора**. Триалы тоже едят время (бесплатно для клиента ≠ бесплатно для
  нас).
- Рост заявок без роста кураторов → нарушения SLA → отток и плохие отзывы →
  деньги на трафик сгорают. **Найм должен опережать закупку трафика.**
- Значит supply-воронку надо строить **параллельно** demand-воронке, с тем же
  вниманием. Ниже — отдельная воронка найма.

---

## 2. Профиль куратора

| Параметр          | Значение                                                           |
| ----------------- | ------------------------------------------------------------------ |
| Статус            | Самозанятый (НПД)                                                  |
| Бэкграунд         | Нутрициолог / диетолог / коуч по питанию / опыт сопровождения      |
| Ключевые качества | Эмпатия, регулярность, неосуждающая коммуникация, дисциплина в SLA |
| Загрузка          | до 14 Pro (или эквивалент Pro+ ~11)                                |
| Чего НЕ требуется | Мед. диагнозы/лечение (HEYS — не медуслуга)                        |

Рынок в нашу пользу: спрос на нутрициологов +34% за год, бум обучения → большой
пул кандидатов, готовых работать удалённо на НПД.

## 3. Воронка найма кураторов

```
Охват кандидатов (каналы ниже)
      ↓  заявка + короткая анкета
Скрининг (бэкграунд, мотивация, тон общения)
      ↓
Тестовое: разбор кейса-рациона + ролевая «клиент сорвался»
      ↓
Обучение по протоколам HEYS (SLA, тон, приложение, анти-срыв)
      ↓
Стажировка на 1–2 клиентах под наставником
      ↓
Полная загрузка (рост до 14 Pro) + бонусы за удержание/SLA
```

### Каналы поиска кураторов

| Канал                                    | Почему                                               |
| ---------------------------------------- | ---------------------------------------------------- |
| profi.ru / Авито услуги                  | 2 000+ нутрициологов онлайн — прямой доступ          |
| Школы нутрициологии (партнёрство)        | Выпускникам нужна практика и доход — win-win         |
| Telegram проф-сообщества нутрициологов   | Тёплая аудитория, низкий CAC найма                   |
| Сарафан/реферал от действующих кураторов | Лучшее качество, культурный фит                      |
| **Конкуренты-нутрициологи 1:1**          | Дать им инструмент + поток клиентов (см. B2B2C ниже) |

## 4. Обучение и стандарты качества

- **Протоколы из брифа:** SLA ≤30 мин в рабочее время; касания утро/обед/вечер;
  отметки 👍/✅/❓/⚠; недельный созвон 20–45 мин.
- **Тон:** неосуждающий, бережный, без «отчётов задним числом», без
  мед-заявлений и запрещённых формулировок (`COPY_VOICE.md`).
- **Шаблоны сообщений** для типовых ситуаций (старт, разбор еды, срыв, чек-ин) —
  снижают нагрузку и держат единый стандарт.
- **Контроль качества (из брифа):** время до 👍 / до ✅, доля нарушений SLA ≤5%,
  покрытие дневника 100%, опросы клиента день 3 / день 7.

## 5. Экономика и удержание куратора

> ⚠️ Связано с блокером W1: при цене Pro 7 990 ₽ старый фикс 7 500 ₽/клиент не
> работает. Рекомендация — **комп как доля выручки** (целевой share ~50%), тогда
> и куратор зарабатывает достойно, и компания в плюсе. Расчёт — лист
> «Кураторы-эконом» в `05_Финмодель.xlsx`.

Мотивация (из брифа, сохраняем как структуру бонусов):

| Элемент                  | Значение                                        |
| ------------------------ | ----------------------------------------------- |
| База                     | Доля выручки за веденных клиентов (реком. ~50%) |
| Минималка                | 50 000 ₽/мес (защита на старте)                 |
| Бонус за удержание       | +500 ₽ за продление клиента                     |
| Бонус SLA                | +5 000 ₽/мес при ≤5% нарушений                  |
| Бонус активный триал     | +500 ₽                                          |
| Бонус конверсия в оплату | +2 500 ₽                                        |

Удержание кураторов (чтобы не выгорали):

- Разумная загрузка (не превышать норму ради краткосрочной выручки).
- **Взаимоподмена** на выходные/праздники (Pro+).
- Снижать нагрузку при падении качества (ограничить новых клиентов) — это уже
  заложено в QA-протокол.

## 6. B2B2C: нутрициолог приводит своих клиентов

Сильный, недооценённый канал роста — **обе стороны воронки сразу**.

**Механика:** действующий частный нутрициолог становится куратором HEYS и
переводит часть своих клиентов на платформу. Он получает инструмент (приложение,
аналитика, протоколы) + долю выручки; HEYS получает и кадры, и готовых клиентов
с доверием.

| Что даёт нутрициологу                   | Что даёт HEYS                                 |
| --------------------------------------- | --------------------------------------------- |
| Приложение + аналитика вместо таблиц    | Клиенты с доверием (низкий CAC)               |
| Снятие рутины ведения дневника          | Готовый квалифицированный куратор             |
| Стабильный доход (доля + бонусы)        | Масштаб supply без агрессивного найма         |
| Поток новых клиентов от HEYS-маркетинга | Защита от оттока (привязка через специалиста) |

**Экономика:** клиент, приведённый куратором, имеет **CAC ≈ 0 по медиа** (платим
только комп куратора как обычно) → лучший LTV:CAC в портфеле. Заложить отдельную
когорту «curator-sourced» в учёте.

**Риск:** привязка клиента к конкретному куратору (уйдёт куратор — уйдёт
клиент). Митигировать: ценность в платформе+бренде, а не только в человеке;
реф-бонусы; плавная передача при уходе.

## 7. Планирование ёмкости (синхронизация с demand)

| Шаг                  | Правило                                                             |
| -------------------- | ------------------------------------------------------------------- |
| Перед ростом трафика | Проверить свободную ёмкость = (кураторы × 14) − активные Pro        |
| Если ёмкости нет     | Список ожидания на триал + ускорить найм, **не** ронять SLA         |
| Найм                 | Опережает закупку трафика на 1 цикл (нанять до того, как лить)      |
| Прогноз              | Лист «Прогноз-12мес» показывает требуемое число кураторов помесячно |

Правило роста (повторяем сквозь весь пакет): **объём маркетинга = функция от
ёмкости кураторов, а не от бюджета.**
