## Как читать

1. Начни с `01` — контекст и факты рынка.
2. `02` — где мы сильны/слабы (там же 2 блокера запуска).
3. `03` + `04` — что и как делаем (стратегия).
4. `06–09` — операционная глубина: конкуренты, плейбуки каналов,
   воронка/онбординг, supply кураторов.
5. `10` — обязательно прочитать перед запуском: что мы могли упустить и что
   делать СЕЙЧАС (легал, оплата, измеримость, этика).
6. `11` — фазовая модель цены: как запускаться основателем-куратором и когда
   поднимать цену.
7. `05` — экономика: «Фазы-цена» (старт vs масштаб), «Реальность-чек» (честный
   LTV:CAC ≈2,7x), «Сценарии»/«Чувствительность» для устойчивости.

## Приоритеты NOW

1. **Запуск Фазы 0** — оффер «первые 15 по стартовой цене с фиксацией»,
   основатель ведёт лично (`11`). Экономика прибыльна сразу (см. «Фазы-цена»).
2. **Юр-структура кураторов (НПД)** — готовить заранее: риск переквалификации; с
   2026 работа с >35 НПД = индикатор проверки. Договоры на результат, не на
   процесс.
3. **Решение по оплате** — ручная помесячная оплата бьёт по LTV. Годовые планы /
   opt-in автопродление.
4. **Измеримость при отключённой аналитике** — промокоды + «откуда узнали» +
   серверная Metrica.
5. **ERID-маркировка + сбор 3%** — до первой закупки платной рекламы (в Фазе 0
   трафик в основном тёплый/органический).

> **Про «блокер W1» (экономика куратора).** На Фазе 0 он не действует: куратор —
> основатель, его время = труд владельца, а не выплата → при цене 7 990 ₽ бизнес
> прибыльный с первого клиента. Блокер превращается в задачу Фазы 1: на сколько
> поднять цену под зарплату внешнего куратора (лист «Фазы-цена», ориентир ~9
> 900–10 900 ₽). Старая цифра «фикс 7 500» из брифа — под устаревшую цену
> 12 990.

## Важная оговорка по цифрам

Цены и операционные параметры (тарифы, загрузка кураторов) — из проекта
(`pricing.ts`, `docs/HEYS_BRIEF.md`). Маркетинговые допущения (CAC, конверсии,
отток, комиссии) — рыночные ориентиры/бенчмарки, помеченные в финмодели как
ДОПУЩЕНИЯ. Их нужно заменить фактом по мере накопления данных за Фазу 1–2.
