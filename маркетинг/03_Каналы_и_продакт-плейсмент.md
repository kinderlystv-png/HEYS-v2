## 1. Позиционирование (одной строкой и развёрнуто)

**Одной строкой:** «HEYS — вас ведут, а не оставляют с цифрами. Дневник ведём
мы, куратор видит контекст недели и помогает вернуться в режим без стыда».

**Категория, в которую заходим:** не «ещё один трекер калорий», а **«личное
сопровождение питания с приложением»** — между бесплатным трекером (нет
человека) и нутрициологом 1:1 (дорого, нет приложения).

**Ревью-слой смены категории:** HEYS должен переносить питание из категории
«контроль калорий» в категорию **«персональное сопровождение режима»**. Тогда
Pro сравнивают не с дешёвым трекером, а с сервисом, который снимает рутину,
держит картину недели и помогает не разваливать режим.

**Три опорных сообщения** (из брифа, разрешённые):

| Сообщение                                                            | Что закрывает           |
| -------------------------------------------------------------------- | ----------------------- |
| «Вас ведут, а не оставляют с цифрами»                                | Отстройка от трекеров   |
| «Дневник ведём мы: вы присылаете фото — мы считаем»                  | Снятие рутины           |
| «Внимательная внешняя опора: вы не остаётесь один на один с режимом» | Эмоциональный мотор Pro |
| «Анти-срыв контур: после отката есть следующий шаг»                  | Доказательство опоры    |

**UTP через категорийный сдвиг**

| Элемент                | Формулировка для HEYS                                                                          |
| ---------------------- | ---------------------------------------------------------------------------------------------- |
| Новая категория        | Персональное сопровождение режима, а не контроль калорий                                       |
| Враг                   | Хаос недели, чувство вины, марафоны и трекеры, которые оставляют человека одного               |
| Кем чувствует себя     | Человеком, который навёл порядок и держит режим при поддержке внимательного куратора           |
| Рациональное основание | Куратор ведёт дневник, видит паттерны и контекст недели, первая неделя без карты, данные в РФ  |
| Guardrail              | Не обещать похудение, отсутствие срывов, медицинский эффект, дружбу или постоянное присутствие |

**Эмоциональная архитектура продажи**

HEYS продаёт не только продуктовую механику, а состояние **внимательной внешней
опоры**. Для клиента это звучит так: «я не остаюсь один на один с едой, усталой
неделей и срывами; есть человек, который держит мой контекст и помогает
вернуться без стыда».

| Эмоция          | Что продаём                                            | Чем доказываем                                                 | Где показывать                                   |
| --------------- | ------------------------------------------------------ | -------------------------------------------------------------- | ------------------------------------------------ |
| Не один         | Есть внимательный куратор, которому важен мой контекст | Куратор ведёт дневник, помнит неделю, пишет в рабочем процессе | Hero, закреп, квиз, первый экран заявки          |
| Без стыда       | Срыв не превращается в обвинение                       | Разбор через режим, усталость и паттерны                       | Telegram-посты, кейсы, бот-результат             |
| Порядок         | Неделя становится понятной, а не хаотичной             | Данные → дневник → паттерны → 1–2 фокуса                       | Лендинг, Pro-описание, день 4 триала             |
| Можно вернуться | Ошибка не ломает неделю                                | Куратор помогает сделать следующий практический шаг            | Анти-срыв посты, кураторские шаблоны, onboarding |

**Guardrail:** эмоция не должна превращаться в обещание дружбы, психологической
помощи, надзора, контроля или постоянной связи. Мы продаём профессиональное
внимательное сопровождение, а не личную зависимость от куратора.

**Нельзя** (юр-риск + тон): «24/7», «психолог», «гарантированно похудеете»,
«вылечу/нормализую сахар», крикливый инфо-стиль марафонов. См. чёрный список в
`COPY_VOICE.md`.

## 2. Аудитория и сегменты

| Сегмент                                  | Боль                                    | Канал захвата                                 | Тариф-вход  |
| ---------------------------------------- | --------------------------------------- | --------------------------------------------- | ----------- |
| **«Срывающийся самостоятельный»** (ядро) | «Знаю как, но не выдерживаю»            | Директ (тёплый интент), Telegram-контент      | Триал → Pro |
| **«Разочаровавшийся в марафонах»**       | «Похудел и набрал обратно, было жёстко» | Контент «почему марафоны не держатся», посевы | Триал → Pro |
| **«Премиум, ценит время»**               | «Нет времени считать, нужен сервис»     | Директ по дорогим запросам, рекомендации      | Pro+        |
| **«Бюджетный самостоятельный»**          | «Хочу контроль, но дёшево»              | Органика, бесплатный контент                  | Self        |

Ядро для платного трафика — первые два сегмента под Pro. Self — апселл-воронка
из органики, не цель платного привлечения (низкий чек не отбивает CAC).

## 3. Каналы привлечения — приоритизация

Принцип: **1–2 канала в фокусе, не распыляться** (узкая команда — W6). Сначала
доказать экономику на одном, потом масштабировать.

### Уровень 1 — запускаем первыми (тёплый спрос + дёшево проверить)

| Канал                             | Зачем                                                                                          | Формат                                  | Маркировка                                   |
| --------------------------------- | ---------------------------------------------------------------------------------------------- | --------------------------------------- | -------------------------------------------- |
| **Яндекс.Директ (Поиск)**         | Ловим готовый интент: «как не срываться с диеты», «нутрициолог онлайн», «не могу похудеть сам» | Поиск → лендинг → триал                 | Авто (ERID)                                  |
| **Telegram (контент-канал свой)** | Прогрев, доверие, органика; главный канал РФ (40% инфлюенса)                                   | Экспертные посты, кейсы, разборы срывов | Свой канал — реклама не нужна; посевы — ERID |

### Уровень 2 — подключаем после первых данных по экономике

| Канал                          | Зачем                                    | Нюанс                                                               |
| ------------------------------ | ---------------------------------------- | ------------------------------------------------------------------- |
| **Telegram Ads / посевы**      | Масштаб охвата по нашей теме             | Только каналы с проверенной статистикой (TGStat); 35–40% — накрутки |
| **VK Реклама**                 | Образовательный контент, добор аудитории | Дешевле Директа на холодную, ниже интент                            |
| **Яндекс.Директ (РСЯ)**        | Ретаргет + look-alike на тёплых          | После накопления пикселя/аудиторий                                  |
| **YouTube / Shorts, VK Видео** | Длинный прогрев, доверие к экспертизе    | Долгий цикл, но дешёвый органический охват                          |

### Не используем

- **Instagram/Facebook реклама** — запрещена с 01.09.2025, штраф юрлицу до 500
  тыс. ₽. Органический аккаунт без цен/ссылок «купить» — максимум для
  присутствия, не канал привлечения.
- **Маркетплейсы (Ozon/WB)** — не наш формат (услуга, не товар).

## 4. Продакт-плейсмент — где и как «вшивать» продукт

Цель — **нативное появление HEYS там, где аудитория уже обсуждает срывы, питание
и усталость от диет**, без крикливой рекламы (тон премиум).

| Тип плейсмента                                       | Механика                                                                                                | Почему работает                                      |
| ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| **Кураторы-нутрициологи как амбассадоры**            | Куратор приводит своих клиентов на платформу; HEYS = его рабочий инструмент + доход                     | O4: бум нутрициологов; органический канал с доверием |
| **Экспертный контент с «бесшовным» продуктом**       | Разборы «почему срывы — это норма и как их ловить» → инструмент HEYS показан как решение, не как баннер | Премиум покупает после ценности, не после крика      |
| **Кейсы клиентов (с согласия, без overpromise)**     | «Как Х перестал срываться по вечерам» — процесс, не «-15 кг за месяц»                                   | Доверие + соответствие `COPY_VOICE`                  |
| **Подкасты/каналы про ЗОЖ, осознанность, выгорание** | Интеграции в нише «забота о себе», а не «худеем к лету»                                                 | Совпадение с тоном анти-срыва                        |
| **Коллаборации с фитнес-тренерами/студиями**         | Тренер даёт тело-нагрузку, HEYS — питание и сопровождение; кросс-реферал                                | Комплементарны, не конкурируют                       |
| **Сравнение-контент «трекер vs сопровождение»**      | Честный разбор, где HEYS — логичный вывод                                                               | Ловит тех, кто уже пробовал трекер и бросил          |
| **Анти-марафонный контент**                          | «Чем системное сопровождение отличается от марафона»                                                    | O3: дискредитация марафонов работает на нас          |

**Правило плейсмента:** продукт появляется как **ответ на боль, уже названную в
контенте**, а не как вставка «а вот и реклама». Везде — реальный SLA и
ограничения (без «24/7», без гарантий результата).

## 5. Воронка под модель куратора

```
Холодный охват (контент/Ads)
        ↓  лид-магнит: разбор/чек-лист «почему срываешься»
Лид (контакт + бот/форма)
        ↓  прогрев в Telegram + квалификация
Заявка на бесплатный триал Pro
        ↓  КУРАТОР назначает дату старта (= лимит = ёмкость кураторов)
Активный триал (≥4 дня, ≥12 приёмов/нед)
        ↓  опыт внимательной внешней опоры + результат недели
Оплата: Self / Pro / Pro+   (Lite/Solo — отдельные гипотезы после Ф0; см. канон 19)
        ↓  удержание (SLA, касания, созвон) + реф-программа
Продление + реферал
```

**Критическая развилка — триал:** это и главный конверсионный актив, и
бутылочное горлышко. Поток заявок на триал **нельзя пускать больше, чем кураторы
вытягивают с SLA ≤5% нарушений**. Маркетинг регулирует кран по ёмкости, не по
бюджету. Лишний спрос → список ожидания (тоже прогрев), а не сорванный SLA.

## 6. Контент-движок (что производим системно)

| Рубрика             | Пример                                            | Цель в воронке            |
| ------------------- | ------------------------------------------------- | ------------------------- |
| Механика срывов     | «Почему вечером рука сама тянется к холодильнику» | Захват боли (топ воронки) |
| Анти-марафон        | «Минус 10 за месяц и плюс 12 потом — почему»      | Отстройка                 |
| Кухня сопровождения | «Как выглядит день с куратором» (без overpromise) | Демонстрация ценности     |
| Кейсы               | История клиента (процесс, не цифры)               | Доверие (низ воронки)     |
| Прозрачность        | «Наш честный SLA и где мы НЕ всесильны»           | Доверие премиум           |

Частота на старте: 3–4 публикации/нед в своём Telegram + 1 длинный материал/нед
(статья vc.ru / VK / YouTube). UGC и короткие видео — приоритет (тренд 2025:
short-video + UGC + performance).
