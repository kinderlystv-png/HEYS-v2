## 1. Текущая консолидированная картина

**Продукт.** Экосистема: приложение + живой куратор + протоколы сопровождения.
Реальный JTBD клиента — **снять тревогу, вернуть контроль, «не быть одному»**;
продаём заботу и анти-срыв, не диету и не счётчик калорий.

**Позиционирование.** Пустая клетка рынка между DIY/AI-трекерами (нет человека,
0–1 500 ₽) и нутрициологом 1:1 (25 000+ ₽, нет приложения): ежедневный человек +
приложение по доступной цене. Данные в РФ (152-ФЗ) — то, чего нет у CalZen.
**Модель-ориентир — Noom** (человек + изменение поведения = миллиардный бизнес):
HEYS — Noom-модель, но RF-native, куратор ведёт дневник за тебя, фокус
анти-срыв. Полный разбор сильных сторон конкурентов — `16`.

**Тарифная лестница (решена; AI — ось, не тариф):**

| Тариф                 | Цена/мес                          | AI                              | Куратор                 | Удержание     |
| --------------------- | --------------------------------- | ------------------------------- | ----------------------- | ------------- |
| Free                  | 0 ₽                               | лимит сканов                    | нет                     | автопродление |
| Solo / AI             | ~990–1 490 ₽                      | безлимит                        | нет                     | автопродление |
| Lite (≈ прежний Base) | ~2 990–3 990 ₽                    | безлимит                        | раз/нед (ревью)         | куратор       |
| Pro                   | ~7 990 ₽ (Ф0; ~9 900–10 900 в Ф1) | «вторая пара глаз», ввод руками | ежедневно               | куратор       |
| Pro+                  | ~14 990 ₽                         | то же                           | ежедн. + тренировки/7-7 | куратор       |

**Фазовая модель запуска.** Ф0 — основатель ведёт лично первых ~8–15 по
стартовой 7 990 ₽ (фиксируется); ведение = прикладной R&D (плейбук куратора +
продукт-бэклог), exit-триггер «плейбук можно передать незнакомцу». Ф1 — внешние
кураторы (НПД) + повышение цены под их зарплату.

**AI-политика.** Solo — AI авто-распознавание = продукт. Pro/Pro+ — еду вносит
человек руками (это его включение в контекст = ров и фича), AI лишь «вторая пара
глаз» (флаги/паттерны → анти-срыв, duty of care). Модель — **RF-native** (Yandex
AI Studio / GigaChat), не зарубежная (152-ФЗ + оплата).

**Удержание следует природе тарифа.** Без человека (Free/Solo) — opt-in
автопродление + напоминания + dunning. С куратором (Lite/Pro/Pro+) — продление
ведёт куратор в нужный момент (+ авто-бэкстоп, опция годовой предоплаты).

**Каналы.** Старт: Яндекс.Директ (горячий интент) + свой Telegram-контент.
Далее: Telegram Ads/посевы (проверка накруток), VK, РСЯ-ретаргет.
**Instagram-реклама — НЕТ** (запрет с 01.09.2025).

**Экономика (headline).** LTV:CAC базовый 5,9x / реальный ~2,7x / цель ≥ 3,0x.
Кэш Фазы 0 при 15 клиентах ~111 000 ₽/мес. Рост = ёмкость кураторов, не бюджет.

**Легал.** Кураторы — самозанятые, ≤5 в год (порог «>35 НПД» не актуален), но
договоры **на результат**, не оклад/график. ERID на всю платную рекламу + 3%
сбор. Мониторить закон о рекламе услуг нутрициологов без медобразования. Тон по
`COPY_VOICE`.

## 3. Решено в методологии ≠ внедрено в продукт (мост в бэклог)

Главный вывод ревизии: бóльшая часть решений живёт в маркетинг-доках, но **ещё
не реализована в продукте/коде**. Это и есть следующий пласт работы:

| Решение                               | Что внедрить                                                                   | Где               |
| ------------------------------------- | ------------------------------------------------------------------------------ | ----------------- |
| Тарифная лестница Free/Solo/Lite      | `pricing.ts` + биллинг (новые планы)                                           | продукт           |
| AI-распознавание еды                  | Cloud Function `heys-api-vision` (RF-модель) + камера в PWA + парсер в дневник | продукт           |
| Автопродление + dunning (Free/Solo)   | ЮKassa рекуррент + напоминания                                                 | продукт/биллинг   |
| Флаг куратору «подписка через N дней» | панель куратора                                                                | продукт           |
| Измеримость                           | промокоды + «откуда узнали» в форме + серверная Yandex Metrica с целями        | продукт/аналитика |
| Договор НПД на результат              | шаблон договора (с юристом)                                                    | операционка       |
| ERID-процесс                          | ОРД VK (бесплатно), регламент отчёта до 30 числа                               | операционка       |
| SEO-калькуляторы (TDEE/BMI/дефицит)   | страницы на лендинге                                                           | продукт/маркетинг |
| Доверие                               | медэксперт/советник + инфраструктура отзывов                                   | операционка       |
| (позже) RuStore/натив, B2B-пилот      | дорожная карта                                                                 | стратегия         |

## 4. Консистентность: что подправлено, что осталось фундаментом

**Подправлено в этом проходе** (синхронизация ранних доков с решениями):

- `01` — к блоку W1 добавлена резолюция (Ф0 снимает блокер) + актуальная
  линейка.
- `02` — W1 переформулирован («снято на Фазе 0»).
- `03` — воронка: «Оплата Base/Pro/Pro+» → «Lite/Pro/Pro+ (Free/Solo —
  self-serve)».
- `04` — добавлен указатель на фазовую модель `11`.
- `12`, `13` — AI в кураторском тарифе = «вторая пара глаз», ввод руками;
  цена-строка.
- `12` — ценовая строка HEYS обновлена под лестницу.

**Осталось как фундамент (валидно, не требует правок):** `01` (рынок РФ), `02`
(SWOT), `03` (каналы/сегменты), `06` (конкуренты), `07–09` (плейбуки, воронка,
supply), `10` (слепые зоны). Это аналитическая база; операционные решения — в
`11–15`.

**Финмодель пересобрана под полную лестницу** (Free/Solo/Lite/Pro/Pro+): unit-
экономика по 5 тарифам (Free показан как −230 ₽/мес — стоимость бесплатного
юзера), CAC/LTV блендед по платящим = **3,97x** (реальный с учётом времени
куратора — **2,81x**), прогноз и сценарии — на блендед-цифрах нового микса.
Технический долг закрыт.
