# Комментарий для модератора RuStore

Вставьте этот текст в поле "Комментарий для модератора" при загрузке версии.

Короткий вариант для поля RuStore с лимитом 180 символов:

```text
Для проверки: Клиент, телефон 70000000002, PIN 1234. Email/MFA не нужны. HEYS - дневник питания с сопровождением, не медицинское приложение.
```

Расширенный вариант, если RuStore даст больше места:

```text
HEYS - это дневник питания и приложение для сопровождения пользователей
куратором. Приложение предназначено для пользователей, которым уже выдан доступ
к HEYS.

Тестовый клиентский доступ для модерации:
Телефон: 70000000002
PIN: 1234

Сценарий проверки:
1. Откройте приложение.
2. Оставьте выбранным режим входа "Клиент".
3. Введите телефон 70000000002 и PIN 1234.
4. Нажмите "Войти".
5. Дождитесь открытия HEYS внутри приложения.
6. В настройках можно проверить ссылки на поддержку, политику
   конфиденциальности, пользовательское соглашение и удаление аккаунта.
7. Выйдите из аккаунта.

Email, пароль и MFA для этого сценария проверки не нужны.
Приложение не ставит диагнозы, не лечит и не назначает медицинские действия.
```

Иконка для витрины RuStore:

- Загружайте `HEYS-rustore-icon-512.png`.
- В APK используется такой же синий знак HEYS Lab.
- Старую зелёную круглую иконку не загружать.

Скриншоты:

- Используйте файлы из `rustore-screenshots-phone-720x1280/`.
- Все скриншоты имеют размер `720x1280`, то есть формат телефона 9:16.
