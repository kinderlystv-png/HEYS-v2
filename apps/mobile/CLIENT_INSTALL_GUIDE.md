# HEYS iOS Client Install Guide

Дата: 2026-07-02

Этот текст используется после App Review и Apple unlisted approval, когда есть
готовая App Store ссылка.

## Инструкция для клиента

1. Откройте ссылку HEYS в App Store: `[ADD_UNLISTED_APP_STORE_LINK]`.
2. Установите приложение HEYS.
3. Откройте приложение и войдите по телефону и PIN, которые вы получили от HEYS.
4. Дождитесь открытия дневника.
5. При желании включите Face ID или Touch ID в настройках приложения.
6. Если уведомления входят в ваш формат сопровождения, включите их в настройках
   iOS после запроса приложения.

Ссылка на unlisted App Store не заменяет доступ к аккаунту. Данные защищены
входом в HEYS, ролями и серверными правами.

## Если что-то не открывается

- Ссылка должна открываться в App Store, а не в TestFlight.
- Для установки нужен Apple ID в регионе, где приложение доступно.
- Если App Store пишет, что приложение недоступно, отправьте скриншот в
  поддержку.
- Если вход не проходит, проверьте телефон/PIN и напишите в поддержку.

Поддержка HEYS: https://t.me/heyslab_support_bot

## Что не нужно делать

- Не устанавливайте `.ipa` файлы.
- Не используйте TestFlight как постоянный способ установки.
- Не пересылайте ссылку как способ дать доступ к данным. Доступ выдаётся только
  через HEYS-аккаунт.
