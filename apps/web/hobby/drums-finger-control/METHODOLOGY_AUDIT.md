# Методологический прогон: пальцы для барабанов

Дата: 2026-06-09.

## Суть

Базовая логика исходного ответа правильная: для барабанного контроля пальцев
нужны rebound/free stroke, finger control, singles, doubles, Moeller/push-pull и
короткие speed bursts. Но исходный текст слишком похож на совет в чате, а не на
продуктовую методику: в нем мало проверок качества, нет прогрессии, нет
разгрузки и есть атрибуции преподавателям без точной ссылки.

## Что оставить

| Блок                      | Вердикт                                         | Почему                                                                                |
| ------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------- |
| Full Stroke / Free Stroke | Оставить                                        | Это база: палка свободно падает, отскакивает, рука не душит rebound.                  |
| Finger Control            | Оставить                                        | Нужен для тихих быстрых нот, ride/hi-hat, singles и контроля второго удара в doubles. |
| Singles RLRL              | Оставить                                        | Основной тест ровности, скорости и смены кисть -> пальцы.                             |
| Doubles RRLL              | Оставить                                        | Второй удар показывает, умеют ли пальцы возвращать палку без зажима.                  |
| Moeller + Fingers         | Оставить, но дозировать                         | Хорош для акцентов и экономии энергии, но новичкам нужен медленный контроль.          |
| Burst Training            | Оставить как режим скорости                     | Работает лучше, когда быстрые отрезки короткие и не ломают звук.                      |
| Buzz Roll                 | Оставить как восстановительный/контрольный блок | Полезен для чувства rebound, но не заменяет singles/doubles.                          |

## Что исправить

| Проблема в исходнике                                  | Исправление для HEYS                                                                                          |
| ----------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| "Только кисть" в Full Stroke звучит жестко.           | Формулировать как relaxed free stroke: движение из кисти, без зажатого хвата, rebound возвращает палку.       |
| "Палка почти не поднимается" может привести к зажиму. | Добавить критерий: низкая амплитуда только если звук ровный, пальцы мягкие, предплечье не горит.              |
| Speed-схема ведет к ежедневным максимумам.            | Максимум контроля не чаще 2 раз в неделю, остальные дни - техника и ровность.                                 |
| Нет критериев перехода BPM.                           | Переход +5 BPM только после 2 чистых минут без зажима, провалов динамики и рассыпания тайминга.               |
| Нет pain/tension gate.                                | Боль, онемение, резкая усталость хвата - стоп. Жжение предплечья > 6/10 - перейти в low_tension_rebuild.      |
| Нет разницы между скоростью и выносливостью.          | Speed = короткие bursts и полный отдых. Endurance = средний темп, длиннее, без максимума.                     |
| Персональные атрибуции слишком точные.                | Не писать "Jojo считает лучшим", если нет точной публичной ссылки. Писать: "в духе школы Mayer/Weckl/Chapin". |

## Продуктовые принципы

1. Сначала звук и расслабление, потом BPM.
2. Пальцы не "качают" силой. Они ловят и направляют отскок.
3. Быстрое движение должно быть маленьким, но не зажатым.
4. Doubles тренируются как два контролируемых удара, а не как случайный bounce.
5. Moeller дает импульс и акцент, пальцы обслуживают rebound между акцентами.
6. Слабая рука получает больше времени: 60/40 или 2:1, если разница заметная.
7. Прогресс считается по clean BPM, а не по пиковому "смог пару секунд".

## Уровни

| Level        | Критерий входа                                    | Что делать                                                             |
| ------------ | ------------------------------------------------- | ---------------------------------------------------------------------- |
| beginner     | Singles < 100 BPM шестнадцатыми чисто 60 секунд   | 15-20 минут, много free stroke и finger rebound, без max days.         |
| intermediate | Singles 100-140 BPM шестнадцатыми чисто 60 секунд | 20-30 минут, balanced + один speed day в неделю.                       |
| advanced     | Singles > 140 BPM шестнадцатыми чисто 60 секунд   | 25-35 минут, bursts, doubles, Moeller phrasing, 2 speed days максимум. |

## Метрики качества

- `cleanBpmSingles16`: максимальный BPM шестнадцатыми 60 секунд без развала.
- `cleanBpmDoubles16`: максимальный BPM doubles 60 секунд без провала второго
  удара.
- `tensionScore`: субъективно 1-10 после сессии.
- `forearmPumpScore`: 1-10 через 2 минуты после сессии.
- `soundEvenness`: self-rating 1-5.
- `weakHandGapBpm`: разница clean BPM между руками на one-hand taps.

## Safety-gates

- Stop: боль, онемение, покалывание, потеря контроля палки.
- Downshift: tensionScore >= 7 или forearmPumpScore >= 7.
- No max: если вчера был speed_breakthrough или сон/восстановление плохие.
- Deload: каждая 4-я неделя или при 3 сессиях подряд с падением clean BPM.
